# Stealth Lookup

Email Data Breach Checker — Terminal Tool

![Terminal Preview](https://img.shields.io/badge/Platform-Linux%20%7C%20macOS%20%7C%20Windows-blue)
![Node.js](https://img.shields.io/badge/Node.js-18%2B-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## Features

- 🔍 Check any email address for known data breaches
- 📊 View breached services, dates, and exposed data types
- 🛡️ Built-in security tips and recommendations
- 🎨 Beautiful colored terminal UI
- ⚡ Real-time breach database lookups
- 🔐 No data stored or logged

## Installation

```bash
git clone https://github.com/ibrahim-dad/stealth-lookup.git
cd stealth-lookup
npm install
```

## Usage

```bash
npm start
```

Or run directly:

```bash
node bin/osint.js
```

### Global Install

```bash
npm install -g .
stealth-lookup
```

## Menu Options

| Option | Description |
|--------|-------------|
| 1 | Check an email address for breaches |
| 2 | Learn how the tool works |
| 3 | View security and safety tips |
| 4 | About the tool |
| 0 | Exit |
